# A-Finance-Manager-App
A Finance Manager Web App Made In Flask With Latest Financial News And Expense Predictions Using ML And AI

intialize the database flask db init command (needed only one time)

migrate the data base by flask db migrate command

upgrade the database by flask db upgrade
 
if getting error try running export FLASK_APP=finance or set FLASK_APP="finance" (for windows)

Run the server by running the run.py file or flask run command in the terminal
